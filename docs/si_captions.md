# SI Figure and Table Captions

## Figures

**Figure S1. Cooperation rates in the pilot experiment.**
(A) Individual cooperation rates by condition and decision-making position. In SM, all participants made independent decisions. In RT, bars show rates for voluntary first movers, second movers after observing cooperation (C), and second movers after observing defection (D). Error bars indicate bootstrapped 95% CIs. Percentages and CIs are displayed above each bar.
(B) Pair-level mutual cooperation rates by condition. Mutual cooperation is defined as both players choosing to cooperate within a pair.

**Figure S2. Decision-time distributions in the pilot experiment.**
(A) Nominal first movers in SM. (B) All decisions in SM. (C) Observed voluntary first-mover decisions in RT. (D) Recovered latent first-mover decision-time distribution in RT, estimated via the two-step procedure described in SI 3. Orange bars represent cooperation; blue bars represent defection. Insets show scaled cumulative distribution functions (CDFs) for each action. In SM, cooperation and defection distributions largely overlap. In RT, defection exhibits a bimodal pattern with an additional peak near the 60-second deadline, indicating strategic delay.

**Figure S3. Decision-time distributions in the main experiment.**
Same format as Figure S2, applied to the main experiment data. (A) Nominal first movers in SM. (B) All decisions in SM. (C) Observed voluntary first-mover decisions in RT. (D) Recovered latent first-mover decision-time distribution in RT. The qualitative patterns replicate: SM shows overlapping distributions, whereas RT defection displays strategic delay near the deadline.

## Tables

**Table S1. Logistic regression: cooperation in SM (pilot).**
Logistic regression predicting individual cooperation in the SM condition of the pilot experiment (N = 74). Predictors: SVO angle, risk aversion, ambiguity aversion, gender, age, and Prolific proficiency (number of completed studies). Continuous predictors are z-scored. Gender is dummy-coded (reference = male). Columns report coefficient estimates, standard errors, z-statistics, p-values, odds ratios, and significance levels. No individual predictor reached statistical significance, though risk aversion showed a marginal negative association with cooperation (b = -0.51, p = .14).

**Table S2. Two-step MLE: voluntary first-mover cooperation in RT (pilot).**
Two-step maximum likelihood estimates predicting voluntary first-mover cooperation in the RT condition of the pilot experiment (N = 78). The model corrects for endogenous first-mover selection using action-specific timing survival functions (see SI 3 for details). Same predictor set and format as Table S1. Higher SVO was significantly associated with greater first-mover cooperation (b = 1.84, OR = 6.33, p = .04).

**Table S3. Logistic regression: second-mover reciprocation in RT (pilot).**
Logistic regression predicting second-mover cooperation after observing first-mover cooperation in the RT condition of the pilot experiment (N = 34). Same predictor set and format as Table S1. No predictor reached significance; estimation was limited by the small sample and near-ceiling reciprocation rate. Ambiguity aversion and gender coefficients were not estimable due to insufficient variance.

**Table S4. Logistic regression: cooperation in SM (main).**
Logistic regression predicting individual cooperation in the SM condition of the main experiment (N = 96). Same predictor set and format as Table S1, with risk and ambiguity aversion measured via bisection tasks. SVO was the only significant predictor (b = 1.01, OR = 2.75, p < .001): more prosocial participants were substantially more likely to cooperate.

**Table S5. Logistic regression: assigned first-mover cooperation in SQ (main).**
Logistic regression predicting assigned first-mover cooperation in the SQ condition of the main experiment (N = 92). Same predictor set and format as Table S4. SVO significantly predicted cooperation (b = 0.73, OR = 2.07, p < .01). No other predictor was significant.

**Table S6. Logistic regression: assigned second-mover reciprocation in SQ (main).**
Logistic regression predicting assigned second-mover cooperation after observing first-mover cooperation in the SQ condition of the main experiment (N = 63). Same predictor set and format as Table S4. SVO was positively associated with reciprocation (b = 0.79, OR = 2.20, p = .01), and ambiguity aversion was negatively associated (b = -0.68, OR = 0.51, p = .05).

**Table S7. Two-step MLE: voluntary first-mover cooperation in RT (main).**
Two-step maximum likelihood estimates predicting voluntary first-mover cooperation in the RT condition of the main experiment (N = 190). Same model specification as Table S2, with the main experiment predictor set. Same format as Table S4. SVO (b = 0.85, OR = 2.33, p < .001), risk aversion (b = -0.53, OR = 0.59, p = .03), and age (b = 0.46, OR = 1.59, p = .04) were all significant predictors. More prosocial and older participants were more likely to cooperate as first movers; more risk-averse participants were less likely.

**Table S8. Logistic regression: second-mover reciprocation in RT (main).**
Logistic regression predicting second-mover cooperation after observing first-mover cooperation in the RT condition of the main experiment (N = 65). Same predictor set and format as Table S4. No predictor reached significance, consistent with the near-ceiling reciprocation rate (93.8%) leaving little variance to explain. The gender coefficient was not reliably estimable due to quasi-complete separation.
